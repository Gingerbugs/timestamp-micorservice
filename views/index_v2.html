Timestamp Microservice

Usage:

  • api/date-string

    Parameters:

    • date-string: date time string

    Returns: JSON object {unix: timestamp, utc: UTC-date-time}

    Example:

    a [project url]/api/2015-12-25

  • api/timestamp

    [project url]/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
